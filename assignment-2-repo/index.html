How to Apply CSS to HTML:

To include an external CSS file in an HTML file, you can use the following code inside the <head> element: 

<link rel="stylesheet" href="style.css">
This further links the external style.css file to your HTML document.

How to Use JavaScript in a Webpage:

To link an external JavaScript file, use the following code inside the <head> or <body> of your HTML code: 

<script src="script.js"></script>
This references the external script.js file and includes it in your webpage.



To write JavaScript directly in an HTML file, you can place your code inside a <script> tag: 
<script>alert("Hello, World!")</script>
This will now run the JavaScript code when the webpage is loaded.
